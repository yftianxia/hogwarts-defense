# 哈利波特塔防 — 游戏概念文档

## 基本信息

- **暂定名称**：霍格沃茨保卫战
- **类型**：2D 俯视视角塔防网页游戏
- **主题**：哈利波特
- **平台**：网页（单文件 HTML + Canvas）
- **技术栈**：纯 JavaScript + Canvas 2D，无需框架

## 核心设定

| 项目 | 内容 |
|------|------|
| 防守方 | 霍格沃茨学生/教授 |
| 进攻方 | 食死徒 + 黑魔法生物 |
| 地图 | 开阔场地（霍格沃茨庭院、魁地奇球场等） |
| 移动方式 | 角色可在地图上自由移动 |
| 操作 | 键盘 WASD 移动 + 自动攻击 + QERF 法术技能 |
| 角色 | 多角色切换（关卡制，每关不同主角） |
| 队友 | 剧情固定，每关不同角色助阵 |
| 节奏 | 剧情模式 + 防守关卡混合 |
| 画面 | 2D 俯视视角 |

## 战斗系统

### 操作
- WASD / 方向键：移动
- 自动攻击：哈利自动向最近敌人发射咒语
- Q / E / R / F：四个法术技能，各有冷却时间

### 技能
| 按键 | 技能 | 效果 | 冷却 |
|------|------|------|------|
| Q | 除你武器 | 金色冲击波，震退周围敌人 + 45伤害 | 3秒 |
| E | 荧光闪烁 | 朝敌人方向锥形光束，眩晕1.5秒 | 8秒 |
| R | 烈火熊熊 | 在最近敌人脚下放火墙，22伤害/秒 | 12秒 |
| F | 统统石化 | 冰冻最近敌人3秒 | 10秒 |

### 敌人类型
| 类型 | 血量 | 特点 |
|------|------|------|
| 食死徒学徒 | 60 | 基础敌人，骷髅面具 |
| 精英食死徒 | 150 | 紫色光环，更肉 |
| 八眼巨蛛 | 500 | BOSS，8条腿8只眼 |

## 关卡设计

### 第一关：城堡大门保卫战
- **场景**：霍格沃茨城堡正门庭院
- **剧情**：哈利刚到霍格沃茨，海格让他守住大门
- **波次**：
  1. 12 学徒
  2. 10 学徒 + 4 精英
  3. 8 学徒 + 6 精英
  4. BOSS：八眼巨蛛幼崽
- **胜利**：消灭全部 4 波敌人
- **失败**：哈利 HP 归零
